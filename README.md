# 五色工具

五色疗法色方生成器 - 基于五色疗法理论的色方生成工具

## 功能特点

- 卦位颜色选择：点击卦位选择对应颜色（绿、红、黄、白、黑）
- 处方生成：自动生成颜色处方和数字+颜色处方
- 图片生成：生成腹部图和左手图
- 处方定义：保存常用处方，方便快速调用
- 响应式设计：适配手机、平板、电脑

## 八卦方位

| 卦位 | 方位 | 角度 |
|------|------|------|
| 艮 | 左下 | 315° |
| 震 | 左 | 270° |
| 巽 | 左上 | 225° |
| 离 | 上 | 180° |
| 坤 | 右上 | 135° |
| 兑 | 右 | 90° |
| 乾 | 右下 | 45° |
| 坎 | 下 | 0° |

## 五色配置

- 绿：#4CAF50 (木)
- 红：#f44336 (火)
- 黄：#FFC107 (土)
- 白：#FFFFFF (金)
- 黑：#000000 (水)
